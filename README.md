# 🎬 VideoHosting

Видеохостинг на Flask с поддержкой загрузки видео, импорта из внешних платформ и системой комментариев.

## 📋 Описание

VideoHosting — веб-приложение, позволяющее пользователям загружать видеофайлы напрямую или импортировать их с популярных платформ (YouTube, VK, Rutube, TikTok, Vimeo) с помощью `yt-dlp`. Поддерживается регистрация пользователей, личный профиль и комментирование видео.

## 🚀 Возможности

- **Регистрация и авторизация** — безопасное хранение паролей через `werkzeug`
- **Загрузка видео** — поддерживаемые форматы: `mp4`, `mov`, `webm`, `mkv`, `avi` (до 2 ГБ)
- **Импорт видео по ссылке** — через `yt-dlp` в фоновом потоке (YouTube, VK, Rutube, TikTok, Vimeo)
- **Лента видео** — главная страница со всеми готовыми видео
- **Профиль пользователя** — список загруженных видео, просмотр профиля других пользователей
- **Комментарии** — возможность оставлять комментарии под видео
- **WAL-режим SQLite** — конкурентная запись без блокировок

## 🛠 Стек технологий

| Компонент | Технология |
|---|---|

## 📦 Установка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/westyyk/videohosting.git
cd videohosting
```

### 2. Создать виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
```

### 3. Установить зависимости

```bash
pip install flask werkzeug yt-dlp
```

### 4. Запустить приложение

```bash
python app.py
```

Приложение будет доступно по адресу: [http://127.0.0.1:5000](http://127.0.0.1:5000)

## 📁 Структура проекта

```
videohosting/
├── app.py              # Основное Flask-приложение
├── tasks.db            # SQLite база данных (создаётся автоматически)
├── uploads/            # Папка с загруженными видео (создаётся автоматически)
├── static/             # Статические файлы (CSS, изображения)
└── templates/          # HTML-шаблоны Jinja2
```

## 🗄 Схема базы данных

**users** — пользователи  
**videos** — видео (локальные и импортированные)  
**comments** — комментарии к видео  

Все таблицы связаны через внешние ключи с каскадным удалением.

## 🔗 Маршруты

| Маршрут | Метод | Описание |
|---|---|---|
| `/` | GET | Главная страница со всеми видео |
| `/register` | GET, POST | Регистрация |
| `/login` | GET, POST | Вход |
| `/logout` | GET | Выход |
| `/upload` | GET, POST | Загрузка видео-файла |
| `/import` | GET, POST | Импорт видео по ссылке |
| `/comment` | POST | Добавить комментарий |
| `/profile` | GET | Личный профиль |
| `/profile/<id>` | GET | Профиль пользователя по ID |
| `/uploads/<filename>` | GET | Отдача видео-файла |

## ⚙️ Конфигурация

| Параметр | Значение по умолчанию |
|---|---|
| Максимальный размер файла | 2 ГБ |
| Допустимые форматы | mp4, mov, webm, mkv, avi |
| База данных | `tasks.db` |
| Папка загрузок | `./uploads/` |
| Порт | 5000 |
